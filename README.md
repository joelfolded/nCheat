## Features

- **Godmode Toggle**  
  Enables or disables godmode by overwriting specific memory bytes.

- **Money Editor**  
  Write custom values directly into memory.

- **Custom Offsets**  
  Uses placeholders like `YOUR_ADRESS`, `YOURBYTE`, and `YOURBYTES` so the user can adapt the tool to any game.

- **Clean GUI (customtkinter)**  
  Dark mode, compact layout, modern design.

- **100% Free & Open**  
  No restrictions, no license limitations, completely public domain.
